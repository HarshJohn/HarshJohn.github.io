Project Overview

This project was a part of Kevin Powell's course on how to make personal portfolio website. Where Kevin provided some .XD files from which I made a clone of the website.
I loved the color theme of that website.
This project allowed me to learn more about CSS-Grid, CSS-Flexbox and Responsive Web Design.
This was also my final project in freeCodeCamp.org's Responsive Web Design Certification program & I was successful in getting my certificate. 😎
I was also able to make a Hamburger Menu with the help of CSS-Pseudo elements, which I've used in index.html page's h2 elements.
